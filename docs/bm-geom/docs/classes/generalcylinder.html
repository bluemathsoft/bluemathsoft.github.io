decompose

  • decompose(): BezierSurface[]

  • Inherited from BSplineSurface.decompose

    • Defined in nurbs/bsurf.ts:1086

    Creates grid of Bezier surfaces that represent this BSpline surface. The routine first computes bezier strips along u (i.e. BSpline surfaces that are Bezier in one direction and BSpline in other). Subsequently decompose it called on each of these strips in the v direction Algorithm A5.7 from "The NURBS Book" See http://www.bluemathsoftware.com/pages/nurbs/funalgo

    Returns BezierSurface[]
